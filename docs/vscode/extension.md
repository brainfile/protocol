# VSCode Extension (Deprecated)

::: warning Deprecated
The Brainfile VSCode extension is no longer maintained.
:::

Brainfile is now built around agent-driven workflows. The recommended alternatives are:

1. **[Pi Extension](/tools/pi)** — Multi-agent orchestration in [Pi](https://pi.dev/)
2. **[MCP Server](/tools/mcp)** — Tool access for Claude, Cursor, and other MCP-compatible agents
3. **[CLI & TUI](/tools/cli)** — Board management from the terminal

For details on the current orchestration model, see the [Pi Extension](/tools/pi) documentation.
