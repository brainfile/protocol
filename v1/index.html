Custom Types

Community types are fully supported! Just extend base.json and provide a schema URL:

---
type: sprint-board
schema: https://mycompany.com/schemas/sprint-board.json
title: Q1 Sprint Planning
---

Renderer Inference

Tools automatically detect how to render data using:

  • Schema hints - x-brainfile-renderer in schema
  • Structural patterns - Detect columns[], entries[], etc.
  • Fallback - Generic tree view for unknown types
